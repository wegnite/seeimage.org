### Fine-Tuning Results

If results aren't perfect, try:

1. **Add specificity**: Include more details about desired outcome
2. **Adjust intensity**: Add "subtle" or "dramatic" to modify strength
3. **Specify preservation**: "while keeping the original facial features"
4. **Quality emphasis**: "with high image quality and sharp details"

## Troubleshooting Common Issues

### Prompt Not Working?

**Check these factors:**
- Is your original photo clear and well-lit?
- Are there multiple people in the photo? Specify which person
- Is the file size appropriate? (Under 20MB works best)
- Are you using Gemini 2.5 Flash model?

### Unexpected Results?

**Try these adjustments:**
- Simplify the prompt by removing complex instructions
- Add "realistic" or "natural" to maintain photo-realism
- Specify "keeping original facial features" for recognition
- Use "professional quality" for better results

### Improving Success Rate

**Best practices:**
- Start with high-quality original photos
- Use specific but not overly complex prompts
- Include style and quality descriptors
- Test with similar photos to find what works
