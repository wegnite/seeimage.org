## 什么是 Nano Banana？

Nano Banana 是 Google 对 **Gemini 2.5 Flash Image** 的昵称，这是他们最先进的图像生成和编辑模型。自发布以来，它已经支持了超过 **2亿次创意编辑**，并为 Gemini 应用吸引了 **1000万+新用户**。

### 核心能力

- **文本生成图像**：从简单或复杂的描述创建高质量图像
- **高级图像编辑**：使用自然语言添加、删除或修改元素
- **风格转换**：应用艺术风格或改变照片的整体美学
- **角色一致性**：在不同场景和姿势中保持相同角色
- **多图像合成**：组合多张图像创建新场景

## 快速入门指南

<Steps>

### 访问 Gemini 2.5 Flash Image

1. 访问 [gemini.google.com](https://gemini.google.com)
2. 选择 **Gemini 2.5 Flash** 模型
3. 上传您的照片或从纯文本生成开始

### 制作您的提示词

使用描述性语言，而不仅仅是关键词。用以下结构组织提示词：
- **主体**：主要焦点是什么
- **风格**：摄影风格、艺术媒介或美学
- **光线**：光线类型和氛围
- **动作**：主体在做什么
- **环境**：设置和背景细节

### 生成和优化

- 生成多个变体
- 使用迭代提示词进行优化
- 比较结果并选择最佳输出

</Steps>

## 提示词工程最佳实践

### 基本原则

**描述场景，不要只列举关键词。** Gemini 的优势在于其深度语言理解能力。叙述性的、描述性的段落几乎总是比不连贯的词汇产生更好、更连贯的图像。

### 摄影模板结构

```
一张写实的[拍摄类型][主体]照片，[动作或表情]，
设置在[环境]中。场景由[光线描述]照亮，
营造出[氛围]的气氛。使用[相机细节]拍摄。
```

### 实例应用
```
一张写实的年轻女性肖像照片，真诚地笑着，头部向后倾斜，
设置在阳光明媚的公园中。场景由黄金时刻的阳光照亮，
创造出温暖的镜头光晕，营造出欢乐的气氛。
使用85mm镜头拍摄，背景有美丽的散景效果。
```

## 高级技巧

### 角色一致性

在多个提示词中保持相同角色：

```
**初始角色定义**
创建一个角色：[外观、服装和独特特征的详细描述]。
在所有后续提示词中使用这个角色设计。

**放置在新环境中**
将上一张图像中的角色放置在[新环境]中，
同时保持他们的确切外观、服装和独特特征。
```

### 精确图像编辑

对现有照片进行特定更改：

```
**背景替换**
保持主体完全不变，但将背景替换为
[新背景描述]。保持原始灯光和透视。

**服装更改**
仅将[人物]的服装更改为[新服装描述]，
同时保持他们的姿势、表情和其他所有内容相同。

**灯光修改**
将此图像中的灯光调整为[灯光描述]，
同时保持所有其他元素完全相同。
```

### 风格转换技巧

```
**艺术风格应用**
将[艺术家/艺术运动]的艺术风格应用到这张照片上，
同时保持核心构图和主题内容。

**摄影风格匹配**
将这张照片转换为匹配[特定摄影师或时代]的摄影风格，
包括他们典型的灯光、构图和后期处理技巧。
```

### 相机和摄影术语

使用特定摄影语言获得更好控制：

```
**相机角度**
- 广角镜头，微距镜头，低角度透视

- 荷兰角度，正面拍摄

**镜头类型**
- 85mm肖像镜头，35mm纪录片风格
- 长焦压缩，广角失真
- 微距镜头细节，鱼眼透视

**灯光设置**
- 柔光箱灯光，环形灯，自然窗光
- 黄金时刻，蓝色时刻，阴天柔光
- 戏剧性侧光，轮廓光，背光
```

## 提示词优化技巧

### 迭代改进

1. **从简单开始**：从基本描述开始
2. **添加细节**：用特定元素增强
3. **优化风格**：调整艺术方向
4. **精细调整**：为完美效果进行小调整

### A/B测试您的提示词

尝试相同概念的变体：
- 不同的灯光描述
- 替代风格参考  
- 各种构图角度
- 多种氛围描述符

### 负面提示

虽然 Gemini 不使用传统的负面提示，但您可以指定要避免的内容：
```
"创建专业肖像，避免过于戏剧性的灯光，
保持自然、真实的表情"
```

## 成功指标和分析

### 优秀提示词的特征

**高成功率指标：**
- 清晰、描述性语言
- 具体但不过于复杂的指令
- 对模型的现实期望
- 正确使用摄影术语

**热门提示词特征：**
- 15-50词最佳长度
- 包含风格、主体和设置
- 使用主动而非被动语态
- 指定氛围和气氛
